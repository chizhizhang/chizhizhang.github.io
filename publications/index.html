Software and Research Outputs

Software outputs listed here reflect tools, engineering workflows, and research interfaces described in the current CV materials.

  1. Aero-Optical Analysis Software
    Python + Qt software for data import, metric calculation, and visualisation for aero-optical analysis, used in internal projects and teaching demonstrations, 2025.
  2. Stress Birefringence Analysis Software
    Python + Qt software for stress-induced birefringence and optical path difference analysis, visualisation, and report export for optical component reliability analysis, 2025.
  3. WTBPerf Blade Performance Assessment Tool
    Python + Qt research utility for blade performance calculation and visualisation, used as a preprocessing tool for wind turbine blade performance and lifetime studies, 2024.
  4. VOX-FE Engineering Simulation and Lifetime Assessment Workflow
    Engineering simulation and fatigue-life workflow developed during postdoctoral research for offshore wind turbine blade assessment, project delivery, and internal validation, 2020-2023.
